## Module 4 - Java I/O Streams

After completing this module, you will be able to:

- Define the concept of an I/O data system, I/O streams, and their various types
- Explain how data is transferred between a program and other external devices
- Identify the various formats of I/O streams
- Use these streams to work with primitive data types and objects
- Apply the interface approach to managing data flows between an application and a file system
- Explain the mechanics of working with archives
- Define the concept of serialization and the mechanics of object serialization/deserialization

## Module 5 - Lambda and Streams

After completing this module, you will be able to:

- Explain the concepts and syntax of functional interfaces and lambda expressions
- Use popular function interfaces such as `Comparator` and general-purpose interfaces from the `java.util.function` package
- Apply lambdas of all types to express program instructions in clear, concise code
- Describe the concept of the Stream API and the main ways to get a stream (generating, collections, and arrays)
- Design and build a stream pipeline with the main intermediate and terminal operations for working with quantities of objects in Java programs
- Create a custom implementation of `Collector` to handle stream elements via the `collect()` method
